morfeus.multiwfn module¶

Multiwfn (http://sobereva.com/multiwfn/) interface code.

Uses pexpect for interactive control.

class morfeus.multiwfn.Multiwfn(file_path, run_path=None, robust_mode=True, debug=False, env_variables=None, max_wait=30, has_spin=None)[source]¶

Bases: object

Multiwfn interface using pexpect for interactive control.

Parameters:

  • file_path (str | Path) – Path to molden/wfn file from xtb or other QM program.

  • run_path (str | Path | None) – Directory for output files. Defaults to temporary directory that is cleaned up when the instance is closed or garbage-collected.

  • robust_mode (bool) – If True, wait for expect patterns between commands (slower but safer). If False, send commands without waiting (faster but may desync).

  • debug (bool) – If True, print debug info including sent commands and expect patterns.

  • env_variables (dict[str, str] | None) – Environment variables to use for the Multiwfn process.

  • max_wait (int) – Maximum time to wait for progress updates in seconds.

  • has_spin (bool | None) – Spin state. If None, detect from file for wavefunction inputs (True=open-shell, False=closed-shell). For .cub/.grd inputs, spin state is left undefined.

get_gaps(n=3, occupation_thresholds=(0.5, 1.5))[source]¶

Return orbital energies and HOMO-centered adjacent orbital gaps.

Parameters:

  • n (int) – Half-window size around HOMO. A window of n=3 spans homo_index - 3 through homo_index + 3.

  • occupation_thresholds (tuple[float, float]) – (lower, upper) thresholds used to classify unoccupied, partially occupied and occupied orbitals.

Returns:

  • orbitals: {index: (energy_ev, occupation, orbital_type)}

  • gaps: HOMO index/energy and sequence_ev, where each entry is {(i, j): gap_ev} for adjacent orbitals i -> j.

Return type:

A dictionary with

Raises:

ValueError – If spin state is unknown, n < 0, or thresholds are invalid.

get_surface(model='esp')[source]¶

Calculate molecular surface properties.

Parameters:

model (str) – Surface analysis model to use. - ‘esp’: Electrostatic Potential - ‘alie’: Average Local Ionization Energy - ‘lea’: Local Electron Affinity - ‘leae’: Local Electron Attachment Energy - ‘electron’: Electron Density - ‘sign’: Sign(lambda2)*rho

Raises:

ValueError – If given model is not available.

Returns:

  • ‘atomic’: Dict mapping atom index (1-based) to atomic properties

    • area_total, area_positive, area_negative: Surface areas (A^2)

    • min_value, max_value: Min/max function values

    • avg_all, avg_positive, avg_negative: Average values

    • var_all, var_positive, var_negative: Variance values

  • ’global’: Dict with global surface statistics

    • surface_area, volume, average, variance, minimum, maximum

Return type:

Dictionary with ‘atomic’ and ‘global’ keys
